订单设置

  • 基本设置
  • 自动确认收货时长
  • 天 商家发货后，达到该时间系统自动确认收货
  • 自动收货提醒时长
  • 天 商家发货后达到该时长，给买家发送即将自动确认收货的短信、邮件提醒
  • 申请退货有效时长
  • 天 买家在该有效时长内需提交退货物流单号及物流公司信息,否则无法进行退货
  • 可评价有效时长
  • 天 到达该有效时长后订单不可评价
  • 可修改评价有效时长
  • 天 订单评价后，到达该有效时长后订单不可修改、删除评价
  • 可追加评价有效时长
  • 天 订单评价后，到达该有效时长后订单不可追加评价
  • 生活购退款有效时长
  • 天 在该有效时长内可以申请退款,0表示不可退款
  • 付款有效时长
  • 小时 提交订单后，在该有效时长内未支付订单则自动取消
