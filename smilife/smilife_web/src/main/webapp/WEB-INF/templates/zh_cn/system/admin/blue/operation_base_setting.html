基本设置

  • 基本设置
  • 是否启用积分
  • 积分启用后，会员注册、登录会获取对应积分，商家商品可以使用积分抵扣，商品购买成功后可以获取积分
  • 是否启用积分商城
  • 积分商城启用后，商城前台自动出现积分商城链接，同时可以管理积分商城商品
  • 是否启用预存款
  • 预存款启用后，会员可以给自己帐户充值进行交易
  • 是否启用金币
  • 金币功能启用后，店主可通过平台提供的交易方式进行购买，金币可用来购买广告、直通车等
  • 金币兑换
  • 一元人民币可兑换 枚金币
  • 是否启用团购
  • 团购启用后，商城前台自动出现团购链接，同时可以管理团购相关信息
  • *团购套餐起价
  • 金币/30天只有购买了团购套餐的用户才可以发布团购，并且团购只能发布团购套餐时间内
  • 是否启用0元试用
  • 开启后可发布0元试用0元试用功能。
  • 是否启用满就送
  • 只有开启满就送才可参加满就送促销方式,自营商品不受影响。
  • *满就送套餐起价
  • 金币/30天只有购买了满就送套餐的用户才可以发布满就送
  • 是否启用满就减
  • 只有开启满就减才可参加满就减促销方式,自营商品不受影响。
  • *满就减套餐起价
  • 金币/30天只有购买了满就减套餐的用户才可以发布满就减
  • *店铺满就减最大数量
  • 审核中或进行中的满就减上限
  • 是否启用组合销售
  • 只有开启组合销售商家才可参加组合销售促销方式,自营商品不受影响。
  • 组合套餐购买价格
  • 金币/30天 商家使用金币购买组合套餐后才可以发布组合销售
  • 组合方案最大数量
  • 商家组合方案数量不能超过该值，最大为10
  • 组合商品最大数量
  • 商家组合数量不能超过该数值，最大为10
  • 可充值金额列表
  • 充值页面上显示的允许充值的金额(单位：元)，以英文逗号分隔，如5,10,20
